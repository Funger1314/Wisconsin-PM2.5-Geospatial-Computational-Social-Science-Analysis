# Wisconsin-PM2.5-Geospatial-Computational-Social-Science-Analysis
Reproducible geospatial analysis of 2024 Wisconsin PM2.5 patterns using 7,234 EPA daily observations, county shapefiles, and Census data. Built with Python, pandas, GeoPandas, SQL, and PySAL, featuring EDA, data cleaning, choropleth maps, Moran’s I, LISA, and methodological validation.
